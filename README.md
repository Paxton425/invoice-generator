# Overview

The *React Invoice Generator* is a modern, responsive single-page application (SPA) designed to simplify the process of creating professional invoices. Built entirely with React and styled beautifully using Tailwind CSS, this tool allows users to dynamically add line items, calculate totals (including tax and discounts), and output the final document for printing or saving as a PDF.

This application is ideal for freelancers, small business owners, and anyone needing a quick, reliable way to manage client billing without the need for complex accounting software.

## ‚ú® Features

Dynamic Item Management: Easily add and remove line items with automatic sequential numbering.

Real-time Calculations: Automatically calculates the line item total, subtotal, tax amount, discount, and grand total as you type.

Configurable Fields: User-editable fields for Invoice Number, Date, Due Date, Tax Rate, and Discount percentage.

Client and Vendor Details: Dedicated sections for entering sender (your company) and recipient (client) information.

Print and Export: Utilizes native browser functionality to generate a print-friendly view, which can be saved directly as a PDF document.

Responsive Design: Optimized layout for seamless use on desktop, tablet, and mobile devices (thanks to Tailwind CSS).

## üöÄ Getting Started

Follow these steps to get a copy of the project up and running on your local machine.

### Prerequisites

You will need the following installed:

Node.js (version 16 or higher recommended)

npm (or yarn/pnpm)

### Installation

Clone the Repository:

_git clone [https://github.com/Paxton425/invoice-generator.git](https://github.com/Paxton425/invoice-generator.git)_
_cd react-invoice-generator_




### Install Dependencies:

_npm install_
# or
_yarn install_




### Run the Application:
Start the local development server. The application will typically run at http://localhost:3000.

_npm run dev_
# or
_yarn start_
